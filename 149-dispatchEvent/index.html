1. 先在 input 框中输入一些文字，会发现通过 v-model 绑定的 text 值会一起改变。

2. 再点击按钮，通过 js 改变 input 的值，会发现 input 的 value 改变了，但 text 并无变化。

3. 这时再任意改变一下输入框中的内容，会发现绑定又回来了

value: {{value}}
text（与 v-model）: {{text}} 
// js 源码
new Vue({
  el: '#app',
  data() {
    return {
      text: '',
      value: ''
    }
  },
  methods: {
    changeValue() {
      this.$refs.input.value = '新的值'
      // 点击后将 input 的 value 值赋值给 this.value
      this.value = this.$refs.input.value
    }
  },
})
